# int数组排序

```java
public int[] sortByBits(int[] arr) {
        arr = Arrays.stream(arr).
                boxed().
                sorted((o1, o2) -> o1 -o2). // sort descending
                mapToInt(i -> i).
                toArray();
        return arr;
}
```
